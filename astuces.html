L'utilité du dossier CSS et IMG

CSS

Le document CSS où il y aura à l’intérieur un document style.css va servir à la mise en page du site. Changer la couleur du texte, à placer les titres etc… ça sera qu’au niveau visuel.

IMG

Le document IMG va vous permettre de stocker toutes les images que vous aurais mis sur votre site
